Live Chat Application's API



  • Source code: https://github.com/zulhaqim1998/chat-app
  • I will deploy this app to heroku as soon as I finish the front-end!

This API are built to clone the chat applications like Messenger. I used socket.io to make the application real-time. This application consists of authorization using token passing (I use JWT). It has the complete routes for signup, login, retreive and sending text messages, just like the real messanging apps. I am still working on the front-end using angular.js and will update it soon.
